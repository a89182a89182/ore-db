  遊戲方式  
  1. 一人限定一個帳號，嚴禁重覆申請！
       開季時系統程式如判斷為多重帳號，將會自動刪除。   
       不接受任何解釋，就算是系統誤判，也不會有補救措施。   
  2. 非該球隊球員，僅能觀察該隊陣容和成績，不能進行比賽，更換棒次，新秀選拔，特殊能力購買。
  3. 參加遊戲的玩家，需完成『 4 天 140 場球季賽 + 總冠軍賽 + 強化日 + 集訓日 』。
  4. 間隔時間：球季賽 10 分鐘﹝連續同一玩家則間隔 15 分鐘﹞。球季賽於 12:00~20:00 縮短為 1 分鐘﹝連續同一玩家則間隔 3 分鐘﹞。
  5. 間隔時間：季後賽 1 分鐘﹝連續同一球隊則間隔 20 分鐘﹞。
  6. 全明星賽在球季第 3 天舉行。球季賽一天可進行 35 場比賽。
  7. 球隊名次的決定依據序為：勝率高 → 勝場多 → 敗場少 → 和局多 → 團隊打擊率高 → 團隊防禦率低 → 運氣好
  8. 球場的距離會影響全壘打，場外全壘打，三壘安打，二壘安打。﹝距離近：本壘打、場外全壘打多，三壘、二壘安打少；遠則反之。﹞
  9. 先發和中繼投手只要表現不差，都會留在場上繼續投，而救援投手只要狀況最差，有機會救援也不會上場。
  10. 球季賽進行中，不得更換野手的守備位置，只能做棒次的更換。
  11. 先發投手為五個投手輪流上場，中繼投手則是三個輪流上場，救援投手則固定不變。
  12. 比賽進行到12局不分勝負，則為和局﹝但是總冠軍賽、季後賽、全明星賽則需要分出勝負﹞。
  13. 強化練習，能力均會提升，除了12年經驗以上，或是年齡35歲以上的球員。總冠軍賽結束後，並無法提升能力。
   參加辦法  
  1. 目前開放名額為 216 名。球季開始後，則無法中途參加比賽。
  2. 高中生(18歲) < 大學生(22歲) < 社會人(23～25歲)，依序球員能力高、年薪及現金多、契約年數短。
   強化練習  
  1. 每球季結束，只有一次強化練習的機會，可以提升能力。
  2. 強化練習後，球員不能退休﹝不包括年齡40歲以上，或有20年經驗以上的球員﹞。
  3. 肌肉練習→提升力量、揮棒練習→提升巧打、跑步練習→提升速度、傳球練習→提升肩力、守備練習→提升守備。
  4. 若無該變化球，變化球練習選「○○系練習」可以學會新球種﹝兩次以上﹞。
  5. 若已經擁有該球種，選「○○○球」則可學到該球種進階的變化球，球種等級則不變。
  6. 若有該變化球，變化球練習選「○○系練習」可以提升變化球的強度。若變化球強度已滿，則無法再提升。
  7. 強化練習時，能力E以下比較容易提升，各項能力由弱到強為：G < F < E < D < C < B < A < S。
  8. 聯盟第六名的球隊：球員強化時有 1/100 的機率碰上可再強化一次的事件。
    聯盟第五名的球隊：球員強化時有 1/200 的機率碰上可再強化一次的事件。
    聯盟第一名的球隊：不會有能力強化事件。例如怪博士改造、名人指導、拾獲秘笈等等。(2013/06/03新增,測試中)
    聯盟前三名的球隊：不會有加薪事件。(2013/06/03新增,測試中)
   球員退休  
  1. 球員需在第三年的球季結束後，球員才能退休，即使還有契約年數也沒關係。
  2. 球員超過40歲後，契約結束就無法再續約。
   球員轉隊  
  1. 如果與球隊的契約已滿，則可以在球季結束後，轉到喜歡的其他球隊。不包括40歲以上，或20年經驗以上的球員。
  2. 如果從上季排名第1的球隊，轉到上季排名第4以下的球隊，球員能力會提升，不會下降。
  3. 如果不是轉到上季排名第4以下的球隊，球員能力有可能會提升，不會下降。
  4. 其他的轉隊情形，球員能力有可能會提升，也有可能會下降。
  5. 如果進行轉隊，現金會依﹝目前球隊上季排名 - 原本球隊上季排名﹞x1000萬增加。
  6. 參考：受限自由球員制度
   球員交換  
  1. 不包括40歲以上，或20年經驗以上的球員，也不包括 電腦 和 明星 的球員。
  2. 契約年數是2～4年，亂數決定。能力的不會提高或下降，年薪會提高300～500萬，亂數決定。
  3. 球季中『契約還有1年』，或球季結束，『契約已結束』的球員，均可以做球員交換，且只有球季結束才能實行。
  4. 如果交換成立，則不能取消。每次提出申請，只限一個對象(申請之後兩天，沒有回應也為交換失敗)。
  5. 參考：受限自由球員制度
   受限自由球員制度  
  1. 在原球隊最近一次離隊記錄必須是引退而非轉隊（且引退當時記錄為年齡40歲以上或年資滿20年），
    才能轉換球隊、提出或接受交換球員。
   球員續約  
  1. 與球隊的契約已滿，願意繼續留在球隊，則可以繼續簽訂契約。不包括40歲以上，或20年經驗以上的球員。
   新秀選拔  
  1. 全明星賽之前：想參加遊戲的玩家，均可參加登記。
  2. 全明星賽之後：在球季結束之前，各隊會從選秀資料裡選擇新球員。
  3. 總冠軍賽之前：新球員會加入球隊﹝但是不參與總冠軍賽的比賽﹞。
  4. 選秀的順位是依照全明星賽之前的戰績做決定。總共進行三輪挑選，各球隊最多可以挑選三名新球員。
  5. 如果玩家人數已超過 216 人，即使有被球隊選上，該球員還是不能加入球隊。
  6. 經由選秀會選拔上的球員，會得到 500 萬的簽約金。
   電腦球員  
  1. 第1年沒有做強化練習、連續2季沒有做強化練習、契約已滿沒有續約的球員，在新球季開幕時會被刪除，由『電腦』代替。
   全明星賽  
  1. 以球季前二天的球員成績，從各個聯盟選出野手9人，先發3人，中繼3人，救援1人，進行3場比賽。另外，明星賽球員狀況通常都是普通。
  2. 從2勝以上的聯盟球隊裡，選出明星賽的ＭＶＰ，野手、投手各1人。
  3. 棒次、守備等全部由程式決定，比賽也是自動進行。球員選拔後，10分鐘後第1場、20分鐘後第2場、30分鐘後第3場及公佈ＭＶＰ。
  4. 明星賽場地：澎湖吉貝巨蛋﹝120m﹞、台東綠島巨蛋﹝110m﹞、宜蘭龜山巨蛋﹝100m﹞。
  5. 明星賽比賽期間，球季賽暫停出賽。

   球隊戰術  
  1. 觸擊‧‧‧觸擊的頻繁度(1← 少 ← 5 → 多 →10)。
  2. 盜壘‧‧‧盜壘的頻繁度，其次考量跑者的速度(1 ← 少 ← 5 → 多 →10)。
   球員能力（野手）  
  1. 力量‧‧‧揮棒的力量，容易成為長打和四死球，三振也變多。
  2. 巧打‧‧‧揮棒的技巧，容易變成安打，不容易打出長打。
  3. 速度‧‧‧跑壘的速度，長打、安打容易出現，但全壘打變少，外野守備影響大。
  4. 肩力‧‧‧傳球的速度，可以阻止對方進壘或阻止盜壘，外野守備影響大。能力高者年薪會增加。
  5. 守備‧‧‧守備的能力，影響失誤的發生，內野守備影響大。能力高者年薪會增加，低者會減少。
   球員能力（投手）  
  1. 球速‧‧‧投手的力量，容易三振打者，也容易被擊出長打，四死球也會變多。
  2. 控球‧‧‧投球的控制，不容易被擊出長打。
  3. 體力‧‧‧投手的體力，隨投球局數減少，被擊出長打或失分也會減少，體力會影響投手的全部能力。
  4. 狀況‧‧‧球員的狀況，容易影響打者得點圈有人和投手危機處理時的能力和體力。

   特殊能力說明  
  1. 同種類的特殊能力只會擁有1種(例如：觸擊Ｘ、觸擊○、觸擊◎，只會有一個)。
  2. 不好的能力可以用買的能力去抵消﹝例如：觸擊Ｘ買觸擊◎，即可以抵消觸擊Ｘ，獲得觸擊◎﹞。
  3. 如果有○的特殊能力，購買同樣◎的特殊能力只需要半價。(例如：觸擊○和觸擊◎)。
  4. 能力不會隨年齡消失，也不能販賣。
  5. 紅色代表不好的能力。
  6. 藍色代表超特殊能力。（此類能力同時間只能擁有一種／新購入的能力將自動取代目前能力）

   夢幻球隊  
  1. 需挑選18名球員，包含9個野手+9個投手(5先發3中繼1救援)，12支球隊都必需有球員被選到，而且每隊最多只能挑選2個球員。
  2. 打擊率(AVG)及防禦率(ERA)最少需要八名『非電腦球員』才會列入該項目的排名，其他項目不拘。
  3. 成績相同時，由該項目平均年薪較低者勝出。
  4. 各個項目獲勝獎金 = [參加人總數] x 1000萬 / 8
  5. 【安慰獎】有任何項目排進前10名即可領安慰獎500萬。每人限一份，不論有幾個項目在前10名都是500萬。
  6. 【參加獎】已經連續五季都有參加夢幻球隊(包含當季)可領參加獎500萬。但是只要中斷就要再重新累計季數。
  7. 安慰獎＆參加獎合計最高1000萬。但是只要有贏得當季夢幻球隊任何一項的第一名獎金即不得再領取安慰獎或參加獎。

  1. 《二獎》該季無人完全猜中6+6+1隊時，只猜中兩聯盟12隊排名，可獲得累計總獎金的 1/10 。
  2. 《三獎》該季無人完全猜中6+6+1隊時，只猜中單一聯盟6隊排名，可獲得累計總獎金的 1/100 。
